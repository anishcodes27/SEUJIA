# Public Assets Folder

Place your static assets here:

## Images to Add

1. **logo.png** - Your company logo (200x200px recommended)
2. **hero-bg.jpg** - Hero section background image (1920x1080px or larger)
3. **favicon.ico** - Website favicon
4. **og-image.jpg** - Open Graph image for social media (1200x630px)

## Usage

Files in this folder are served from the root URL.
